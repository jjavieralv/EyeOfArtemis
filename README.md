# EyeOfArtemis
Complete guide about how to create a complete infrastructure (telecommunications antennas, network software, server software, and IA) to install in the forest and enables you to monitor with alerts and metrics animals, people, cars... It also detects fire and smoke in order to avoid wildfires alerting quick emergency authorities to extinguish it before it grows.
 Everything uses open-source software and is free to use. 
Let's reduce wildfires!
